# Caterer System

## Installation

Clone the repository and move to backend folder and install dependencies.

```bash
npm install
```

Move to frontend folder and install dependencies.

```bash
npm install
```

---

## Running the Application

Go to the backend project directory and start the server using Nodemon.

```bash
npm run dev
```

Server will start on:

```text
http://localhost:5000
```

---

Go to the frontend project directory and start the server using

```bash
npm run dev
```

Server will start on:

```text
http://localhost:5173
```
